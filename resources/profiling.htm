Profiling

Profiling is a process where you use a Profiler (software) to analyze the speed of a program. We will use the Profiler that is part of Developer Tools in Chrome.

Capturing a profile

When you run a Profiler it will capture (record) a Profile, which is a recording of what went happened as the program ran. The Profiler can then use the information in the Profile to show you what functions took the most time to execute.

To capture a profile, follow the following steps:

  • Load your program into Chrome

  • Open Developer Tools

  • Click on the Profiles tab, and make sure "Collect Javascript CPU Profile" is selected.

  • Click the Start button to start capturing the profile.

  • Use your program the way you think a user would use it.

  • Click the Stop button to stop the capture.

  • Chrome will show you the profile.

    The profile information will contain functions that you wrote as well as functions that you did not write. For example, in the bubbles example, the function drawFrame function is listed, but the canvas functions like fill an arc are also listed.

  • The Self column is the percent of time taken by the function not including any functions it called.

  • The Total column is the percent of time taken by the function including any functions it called.
