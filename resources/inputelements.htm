Input Elements

Text (single line)

An element for entering a single line of text.

<label>
    planet name:
    <input type="text" value="saturn" id="planet">
</label>

Text (multiple lines)

An element for entering one or more lines of text.

<label>
    description:<br>
    <textarea id="description" rows="5" cols="50"></textarea>
</label>

Number

An element for entering a number.

<label>
    planet size:
    <input type="number" value="123" id="planetSize">
</label>

Password

An element for entering a password (or other confidential information).

<label>
    enter your PIN code:
    <input type="password" value="123" id="pin">
</label>

Range

An element for selecting a number between a range of numbers.

<label>
    temperature:
    <input type="range" value="30" min="0" max="100" id="temperature">
</label>

Color

An element for selecting a color.

The value has to be color in hex. You can't use named color values like "orange".

<label>
    hair color:
    <input type="color" value="#ff8000" id="hair">
</label>

Checkbox

An element for selecting an option.

<label>
    <input type="checkbox" value="on" id="bacon">
    include bacon
</label>

Radio

An element for selecting from a small set of options.

<label>
    <input type="radio" checked="checked" id="mayonaise" name="spread">
    mayonaise
</label>
<label>
    <input type="radio" id="baconnaise" name="spread">
    baconnaise
</label>
<label>
    <input type="radio" id="none" name="spread">
    none
</label>

Select

An element for choosing from a larger set of options.

<label>
    cat to adopt:
    <select id="adopt">
        <option value="cat1">Paws</option>
        <option value="cat2">Whiskers</option>
        <option value="cat3">Scruffy</option>
        <option value="cat4">Spots</option>
        <option value="cat5">Tabs</option>
    </select>
</label>

Button

An element for selecting an action.

<input type="button" value="Place Order" id="order">

Canvas

The canvas element can be used for mouse input. If you want the canvas to allow key input, you can use the tabIndex property.

<canvas width="200" height="200" id="mainCanvas" tabIndex="0"></canvas>

Input Events

Handling events

You have to have a different function for each event you want to handle.

The following shows how to handle various events for an element.

function start() {
    // find element
    var someElement = document.getElementById("someid");

    // handle mouse events
    someElement.onmousemove = someMouseFunction;
    someElement.onmousedown = someMouseFunction;
    someElement.onmouseup = someMouseFunction;
    someElement.onclick = someMouseFunction;

    // handle touch events
    someElement.ontouchstart = someTouchFunction;
    someElement.ontouchmove = someTouchFunction;
    someElement.ontouchend = someTouchFunction;

    // handle key events
    someElement.onkeydown = someKeyFunction;
    someElement.onkeyup = someKeyFunction;
    someElement.onkeypress = someKeyFunction;

    // handle change event
    someElement.onchange = someChangeFunction;
}

Mouse events

The following is an example function that can handle onclick, onmousedown, onmousemove, or onmouseup events.

function someMouseFunction(event) {
    var x = event.clientX;
    var y = event.clientY;

    // do something here
}

Touch events

The following is an example function that can handle ontouchstart, ontouchmove, or ontouchend events.

function someTouchEvent(event) {
    // get all the touches (all fingers)
    var touches = event.targetTouches;    
    // get the first finger location
    var firstTouch = touches[0];
    // get the location of the finger
    var x = firstTouch.pageX - event.target.offsetLeft;
    var y = firstTouch.pageY - event.target.offsetTop;

    // do something here
}

Key events

The following is an example function that can handle onkeydown, onkeyup, or onkeypress events.

function someKeyFunction(event) {
    // get key code from event
    var keyCode = event.keyCode;

    // do something here
}

Change event

The following is an example function that can handle onchange events.

When a change event occurs, you will typically want to get the value of the element that changed.

function someChangeFunction(event) {
    // get the element that changed
    var element = document.getElementById("someid");

    // get the new value
    var value = element.value;

    // do something here
}

Input Properties

The following properties change when there is input.

Value

To get or set the value property, do the following:

function someFunction() {
    // get the element
    var element = document.getElementById("someid");
    
    // get the value
    var value = element.value;
    
    // change the value
    element.value = "new value";
}

Checked

To get or set the checked property, do the following:

function someFunction() {
    // get the element
    var element = document.getElementById("someid");
    
    // get the value
    var isChecked = element.checked;
    
    // change the value
    element.checked = true;
}
