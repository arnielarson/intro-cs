Bootstrap

Bootstrap is a CSS framework created by Twitter that takes makes HTML "look nice" and "behave nicely" with less effort on your part. Here are some of the advantages:

  • The Bootstrap styles are much nicer than "plain" HTML, meaning you will have to create fewer styles.
  • Bootstrap has a "grid" (explained below). A grid is an easy way of saying "what goes where". A nice advantage to the Bootstrap grid is that it works nicely on both desktop and mobile devices (meaning the page will automatically adapt to larger and smaller screens).
  • Bootstrap contains additional components that are useful for creating menus, etc.
  • Bootstrap has already figured out what colors to use for different situations: warning a user, in case of a problem, etc.

Getting Started

To use Bootstrap, do the following:

  • Download bootstrap from http://getbootstrap.com. Choose the "Download Bootstrap" option (not Download Source).
  • Unzip the contents of the download. You should see a dist folder.
  • Copy the contents of the dist folder into the folder for your program. You should have three folders in your program folder: css, fonts, and js.
  • Use the following template for your HTML pages:
    <!DOCTYPE html>
    <html>
        <head>
            <title>Page Title</title>

            <!-- bootstrap stuff -->
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link href="css/bootstrap.min.css" rel="stylesheet">
            <script src="https://code.jquery.com/jquery.js"></script>
            <script src="js/bootstrap.min.js"></script>

            <!-- your stuff -->
            <script src="uprep.js"></script>
            <link href="uprep.css" rel="stylesheet">
        </head>
        <body onload="start();">
            <!-- you will decide what goes here -->
        </body>
    </html>

The rest of this document id going to discuss what goes inside the body element.

Adding a Grid

A Grid is a way of deciding "what goes where" on a page. A Grid is arranged into rows and columns. Each row can have its own set of columns. Each column takes up a specific amount of space on the page, measured in units of 12. Each column can take up between 1 and 12 units of width.

The width of all the columns in a row should add up to 12.

Lets look at an example. In this example, we will assume the following styles have been defined in uprep.css which set the background color. The purpose of these styles is just so we can see the areas that are taken up by each part of the document.

.red {
    background-color: red;
    color: white;
}

.green {
    background-color: green;
    color: white;
}

.blue {
    background-color: blue;
    color: white;
}

This example has three rows. The rows contain paragraphs but we could have used any element.

  1. The first row has a single column that is 12 wide (the full width of the grid).
  2. The second row has three columns that are all 4 wide (the full width of the grid).
  3. The third row has two columns. The first column is 4 wide and the second is 8 wide.

You can try the example here. Try resizing the Chrome window horizontally and you will see how the grid automatically adjusts.

<div class="container">
    <div class="row">
        <div class="col-xs-12">
            <p class="red">
                This is the first row (12)
            </p>
        </div>
    </div>            
    <div class="row">
        <div class="col-xs-4">
            <p class="red">
                Second row, first column (4)
            </p>
        </div>
        <div class="col-xs-4">
            <p class="green">
                Second row, second column (4)
            </p>
        </div>
        <div class="col-xs-4">
            <p class="blue">
                Second row, third column (4)
            </p>
        </div>
    </div>            

    <div class="row">
        <div class="col-xs-4">
            <p class="red">
                Third row, first column (4)
            </p>
        </div>
        <div class="col-xs-8">
            <p class="green">
                Third row, second column (8)
            </p>
        </div>
    </div>        
</div>
