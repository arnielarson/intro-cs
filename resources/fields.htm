Example <form> fields

The following some examples of elements used to construct fields within forms.

For more information and additional options, see the HTML5 specification:

<input>

Select the type of input using the "type" attribute. Below are example of different input types.

radio

<textarea>

A textarea can be used for multi-line text.

<select>

A select can be use to let the user select from a set of options.

<fieldset>

A fieldset can be used to name and visually group a set of fields.
