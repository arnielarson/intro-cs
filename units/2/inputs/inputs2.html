Launching a bird using the mouse

In order to launch the bird with the mouse/trackpad, we need to handle THREE events:

  1. When mouse button is pressed down (onmousedown event) on the canvas (id is "mainCanvas"), do this:

    putBirdInRubberBand(x, y);

    This will put a new bird into the rubber band.

    You will need to get the x and y from the event object.

    When you click in the canvas, as bird should be added to the rubber band where you click, BUT... when you drag or let go, NOTHING will happen.

  2. When mouse is moved (onmousemove event) over the canvas (id is "mainCanvas"), do this:

    moveRubberBand(x, y);

    This will redraw the rubber band as you move the mouse.

    You will need to get the x and y from the event object.

    You can now click and drag and the rubber band should follow your movement, BUT... when you let up NOTHING will happen.

  3. When mouse button is released (onmouseup event) over the canvas (id is "mainCanvas"), do this:

    releaseRubberBand();

    This will actually launch the bird, and will make space for a new one.

    You should now be able to launch birds with your mouse.

The "Start Over" button

To make the "Start Over" button work, we need to handle one event:

When the mouse button is pressed and released (onclick event) on the "Start Over" button (id is "start"), do this:

startGame();

This is used to start or restart the game.

When you press the "Start Over" button, a new set of pigs should appear.

The "Launch Bird" button

To make the "Launch Bird" button work, we need to handle one event:

When the mouse button is pressed and released (onclick event) on the "Launch Bird" button (id is "launch"), do this:

launchBirdAtAngleAndSpeed(angle, speed);

This will load a bird and launch it, given a speed and an the angle (in degrees).

You will need to get the angle and speed values from the text elements. The id's of the elements are "angle" and "speed".

When you press the "Launch Bird" button, a bird should launch with correct angle and speed.
