3.2.1 - Setup Nitrous

Nitrous

Follow the instructions below to set up Nitrous, which is a way to create web applications using a web browser.

Please read and follow the instructions carefully.

Go to Nitrous

Use Chrome to go to http://www.nitrous.io.

Sign up

Enter your nitrous user name. Use your uprep user name followed by -uprep (for example, echristensen-uprep).

Enter your uprep email address and a password and click sign up. Remember your password!

Confirm your account

An email will be sent to your uprep email account. Follow the instructions in the email.

Create a Box using Node.js

After clicking on the link in the email, you should be taken to a screen that lets you create a "Box".

Nitrous calls a web server a "Box".

Select as the type of box (see picture below). Make sure you do this!.

Leave the Name and Region as they are. The name is the name of the server in nitrous and will not show up in the URL for your application. We will give you a chance to pick a URL for your web application (including a DNS host name) later in the project.

Click Create Box.

Nitrous runs on "N2O", which are units of processing that you consume by using Nitrous. If you run out, talk to your teachers (and we will add some). If you get emails from Nitrous about "almost running out of N20", you can ignore them.

Nitrous has a web application (which you can see by pressing "Take my to my Box"), but we will use the Chrome Application instead. The Chrome Application is very similar but it has some nice features like copy and paste.

Install the Nitrous Chrome App

Click the Store icon in Chrome.

Search for nitrous.

You will see two versions of the nitrous application. You want version 1.0.2. Click on the first one and select the details tab.

You will see the version at the bottom right of the screen.

You might need to go back and look at the second one to find the right version (1.0.2).

Once you have found the right version, install it correct version of the app by clicking the button.

Chrome may ask you for your password. Use the same username/password you use for Google Apps at University Prep.

Nitrous will appear as an application along with the rest of your Google apps.
