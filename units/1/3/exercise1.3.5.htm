The HTML of an element

Many HTML elements can contain other elements. For example, the body element below contains 4 other elements:

<!DOCTYPE html>
<html>
    <head>
        <title>Title</title>
        <link rel="stylesheet" href="pets.css">
        <script src="pets.js"></script>
    </head>
    <body class="main" onload="start();">
        <h1 class="section">Cats</h1>
        <p class="detail">123</span></p>
        <h1 class="section">Dogs</h1>
        <p class="detail">456</p>
     </body>
</html>

How we change these elements from our Javascript?

It turns out that each element object that can contain other elements has a property called innerHTML. The innerHTML is a string value for all the HTML, shown in yellow above.

Changing the innerHTML of an element

If we wanted to change the highlighted HTML above to the following:

<!DOCTYPE html>
<html>
    <head>
        <title>Title</title>
        <link rel="stylesheet" href="pets.css">
        <script src="pets.js"></script>
    </head>
    <body class="main" id="change" onload="start();">
        <h1 class="section">Weasels</h1>
        <p class="detail">999</h1>
     </body>
</html>

We can change the value of the innerHTML property of the Body Element. Of course, we would first name the body element so we can find it (we'll name it pets in this example).

<!DOCTYPE html>
<html>
    <head>
        <title>Title</title>
        <link rel="stylesheet" href="pets.css">
        <script src="pets.js"></script>
    </head>
    <body class="main" id="pets" onload="start();">
        <h1 class="section">Cats</h1>        
        <p class="detail">123</span></p>
        <h1 class="section">Dogs</h1>
        <p class="detail">456</p>
    </body>
</html>

Then we can write:

function start() {
    var body = document.getElementById("pets");
    body.innerHTML = "<h1 class=\"section\">Weasels</h1><p class=\"details\">999</p>";
}

You will notice that writing HTML in Javascript is a little tricky. Each time we use a double quote in a string, we have to write \". You can find more information here.
