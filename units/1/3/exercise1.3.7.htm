Bootstrap

You might have found that it is difficult to come up with styles that look nice. I wish there were some styles I could just use...

Bootstrap is a set of CSS files (and images, and javascript) made by Twitter that are "ready to use" to build web pages. To try them out, do the following:

  • Download the following zip and extract the files.

  • Open default.html in Chrome and you should see:

  • Edit default.html in Komodo to use different styles from Bootstrap.

Here are some ideas for what you can do:

  • Learn more about the hero unit style shown above.

  • Put some nice borders around your images.

  • Learn more about all the elements that are used for text here.

  • Add some icons to your page.

  • Add some buttons to your page. You can use the onclick attribute to call a function in Javascript button is clicked.

  • Need some navigation? dropdowns? Button groups? Alerts (the good kind)? no problem.

  • Add a grid to your page, and put some elements inside the grid (inside the div elements). A grid is a way of dividing the page up into parts so that it works nicely when resized.
