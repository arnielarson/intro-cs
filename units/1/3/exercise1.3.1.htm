Cascading Style Sheets (CSS)

Do your pages have style?

When we write a paragraph element in HTML:

<p>
    University Prep Pumas!
</p>

We are saying what we want displayed: a paragraph with the words "University Prep Pumas!". But we aren't saying how we want it to look: that is the job of CSS.

For example, here is the above paragraph with three different CSS styles:

University Prep Pumas!

University Prep Pumas!

University Prep Pumas!

CSS is not limited to paragraphs. Most elements on the page can be styled with CSS.

How does CSS work?

CSS is another resource that you use from your HTML page. This means you need to create a CSS file (ending in .css) and link it to your HTML file in order to use the styles in the CSS file. This is very similar to how you include a Javascript file in your HTML.

<!DOCTYPE html>
<html>
    <head>
        <title>University Prep</title>
        <link rel="stylesheet" href="uprep.css"></link>
        <script src="uprep.js"></script>
    </head>
    <body>
    </body>
</html>

What is in a CSS file?

A CSS file is full of styles. Each style has a different name, and you get to choose the name.

When we write an HTML element, we can say which style to use for the element by giving its name (using the class attribute).

So how do we write a style?

CSS is a language for describing styles. What does that language look like? Lets look at the CSS styles for our example:

style1

.style1 {
    border-style: solid;
    border-width: 8px;
    border-color: #036355;
    padding: 10px;
    color: white;
    font-size: 22pt;
    background-color: black;
    font-weight: bold;
    font-family: monospace;
    background-image: url("inside-uprep.png");
}

style2

.style2 {
    padding: 10px;
    border-style: solid;
    border-width: 1px;
    border-color: rgb(0, 87, 132);
    background-color: white;
    color: rgb(0, 87, 132);
    font-size: 24pt;
    font-family: "Kepler Std Light 1166513", Rockwell,
        Georgia, "Times New Roman",
        "Bitstream Vera Serif", serif;
}

style3

.style3 {
    text-align: center;
    font-weight: bold;
    font-style: italic;
    border-radius: 10px;
    padding: 10px;
    background-color: green;
    color: white;
    font-size: 18pt;
    font-family: Georgia;
}

See if you can match any parts of the CSS style language with what the style looks like.

What are the rules for the CSS language?

Although it may look complicated, the CSS language has only a few concepts:

  • A CSS file contains a set of styles, one after another.
  • Each style has a style name, and a set of properties.
  • Each property has property name and a property value.
  • Properties with different names control different parts of the style.

For example, a style with three properties will always look like this (only with the      filled in):

.style name {
    property name: property value;
    property name: property value;
    property name: property value;
}
Another language, and another set of rules! CSS is pretty picky, so...
  • Remember the period! .
  • Remember the colon! :
  • Remember the semicolon! ;
  • Remember the braces! { and }
