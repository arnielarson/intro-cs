Exercise 1.5.4- A harder problem with Arrays

Calculating Prime Numbers

Prime Numbers are numbers that are only divisible by 1 and themselves.

Here are some examples:

2
3
5
7

7 is a prime number because it is only evenly divisible by 1 and 7.

In this assignment you will create a program that generates prime numbers!

To do this, you will need to think of a way to write a computer program to check if a number is prime.

You will need to create an algorithm. You might want to read about the Sieve of Eratosthenes - an algorithm that could be used.

The Assignment

  • Create a program that can calculate prime numbers

  • Use your knowledge of for loops and arrays to help you generate prime numbers

  • For each prime number you find, you should probably keep track of it in an array

  • Tell your instructor what the 100th prime number is

Done!

When you've finished this example, show your instructor how your program works.
