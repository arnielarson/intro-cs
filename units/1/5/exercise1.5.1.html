Summing an Array

Now that you understand that arrays are another useful datastructure for computer programming lets get some more practice with them.

In this exercise you will use an array and a for loop to calucate the sum, just like in the previous example.

For example, if your array had the values 6, 2 and 3

var numbers = [ 6, 2, 3 ];

Then the results of your caluclation would be 11.

var sum = 6 + 2 + 3 = 11;

Just like in the example, when you calculate your sum, you should use a for loop to "look" at each element in the array.
