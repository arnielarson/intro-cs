Exercise 1.5.3- Methods with Arrays

Array Methods

In this exercise, you should investigate some of the methods that arrays have!

You've already seen one property that arrays have, the length property

var myarray = [ 1, 3, 7, 11 ];
var length = myarray.length;

Arrays are also Javascript objects! They have other methods associated with them. A few of those methods are show below.

For this exercise you should try out all of the methods listed below.

The Sort Method

Arrays have a sort method. When this method is called, the elements of the array are sorted and rearragned in order.

var numbers = [ 5, 4, 3 ];
numbers.sort();
var firstElement = numbers[0]

What value does the variable firstElement have now?

The Push Method

Arrays have useful methods which can add and remove elements.

The push method can be used to add a value at the end of an array.

var planets = [
    "Earth",
    "Mars",
];
planets.push("Venus");

Now what is the value of planets?

The Pop Method

Arrays have a useful methods to remove elements too.

The pop method can be used to remove the first element of the array.

var planets = [
    "Earth",
    "Mars",
    "Venus",
];
var planet = planets.pop();

What will the var planet be after this code runs?

The Splice Method

Arrays have another useful methods for adding and removing elements from any point in the array.

The splice method is used to remove one or more elements from an array, and can also be used to add elements to an array.

var planets = [
    "Earth",
    "Mars",
    "Venus",
];
// at index 1, remove 1 element
planets.splice(1, 1);
// at index 1, add a new elements
planets.splice(1, 0, "Saturn");

What values does the planets array now contain?
