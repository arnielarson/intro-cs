Slope of a line using objects

To practice working with objects, lets update the program that calculated the slope of a line in the last lesson (Exercise 1.2.4) to use objects.

Points of a line (x1, y1) and (x2, y2) are an obvious place to start when grouping together variables.


Lets create Point objects for our program which calculates the slope of a line.

The assignment
