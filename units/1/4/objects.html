Objects

We have learned that in Javascript, Objects are provide a way to group variables and functions.

When we use an object, we call the variables it contains its properties and the functions it contains its methods.

One object that we've created so far was a year object, which simply grouped the data that we were using in our program in exercise 1.3.4 to display the number of students at UPrep.

Writing objects

In Javascript, an object is a value (just like a number is a value).

The way we write an object value in Javascript is as follows:

{ properties }

Each property is separated by a comma (,), and is written:

name: expression

For example, for our year object, we can write an object value with seven properties:

{
    grade6: 66,
    grade7: 74,
    grade8: 74,
    grade9: 82,
    grade10: 81,
    grade11: 79,
    grade12: 76
}

Using properties

So far we have seen how to write a object value, and what happens when we place an object in a variable. How do we actually use the values contained in an object?

To use the value of a property, we need to answer two questions:

  • Which object?
  • Which property?

To use the value of a property of an object, we write:

object.property

Where object refers to an object value, and property is the name of the property we want to use.

For example, say we wanted to total the number of students in the middle school for this year. The object value we want to use is referenced by variable thisYear, and we want to use the values of the properties named grade6, grade7, and grade8. We can then write:

var middleSchoolStudents = thisYear.grade6 + thisYear.grade7 + thisYear.grade8;

Objects as parameters

Objects can also be passed to functions as parameters.

For example, the following defines a function (countMiddleSchoolStudents) that will calculate the total number of students in the middle school for any year object. We can call that function from another function that calculates the number of middle school students for this year, and provide the year object we want counted as a parameter.

function countMiddleSchoolStudents(year) {
    return year.grade6 + year.grade7 + year.grade8;
}

function calculateStudents() {
    var thisYear = {
        grade6: 66,
        grade7: 74,
        grade8: 74,
        grade9: 82,
        grade10: 81,
        grade11: 79,
        grade12: 76
    };
    
    var middleSchoolStudents = countMiddleSchoolStudents(thisYear);
}

Objects as return values

Functions can only return a single value. However, values can also be objects. And a function can also return an object.

Can you see how this coule be used?

Here's an example of function returning an object.

function makeBubble() {
    var bubble = {
        x: 50,
        y: 50,
        size: 10,
        color: "red",
    }
    return bubble;
}

Look at this bubble example to see if you can see why a bubble object might be useful.

New Syntax and Vocabulary

Although using objects in Javascript is pretty simple, since we're just grouping variables and functions, there will be new Javascript keywords and syntax that we'll have to learn.

There are two new keywords that we'll learn

  • The this keyword is used to refer to the properties within an object

  • The new keyword is used to create an object

We'll see examples of how these keywords are used in Javascript over the course of this lesson.

Methods

As you know, when you add a function to an object its called a method, but how do you actually add a function?

Here's an example of a method that changes the color of our bubble object:

var bubble = {
    x: 50,
    y: 50,
    size: 10,
    color: "red",
    changeColor: function(newColor) {
        this.color=newColor;
    }
}

Object methods can be used in the same way that properties are called

object.method()

To use the bubble changeColor method we can easily call it and pass in a new color as a parameter to the function:

var color = "blue"; bubble.changeColor(color);

More Objects

The way that we've been writing objects is not the only correct syntax in Javascript!

There is another way to write objects which is sometime called Constructor Syntax

The bubble object below is created using constructor syntax. What do you notice?

  • The bubble is created with the function keyword

  • The bubble properties and methods use the this keyword

  • The bubble properties and methods use the = operator for assignment

  • The bubble properties use ; instead of ,

  • New bubble's are created with the new keyword

function Bubble() {
    this.x = 50;
    this.y = 50;
    this.size = 10;
    this.color = "red";
    this.changeColor = function(newColor) {
        this.color=newColor;
    };
}

var myBubble1 = new Bubble();
var myBubble2 = new Bubble();
