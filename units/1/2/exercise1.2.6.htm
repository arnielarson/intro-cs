Area of a triangle

Programming is not all about typing in an editor. Sometimes you have to spend time finding a solution to a problem, or looking up which function to use. This exercise will require more investigative work. Use the web as your resource.

Create a function that calculates the area of a triangle given the three points that make up a triangle.

Create another function to test your function with at least two different triangles.

Remember you used Math.floor in the last exercise? You may need to use additional functions from Math in this exercise.
