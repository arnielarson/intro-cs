How Many Coins?

If you buy something at a shop, and you give the shopkeeper too much money, they will (hopefully) give you some change back.

In this exercise, we will create a function that calculates how many coins a shopkeeper will give as change.

We will assume that the shopkeeper only gives change using the following coins:

Name of
Coin		Cents per
Coin
Quarter25
Dime10
Nickel5
Penny1

Also, the shopkeeper would rather give you a large coin rather than a small coin.

The input to your function should be the amount of change in cents. The output should be the number of coins.

What you need to know

There is more than one solution to this problem. But you will probably need to know how to do at least one of the following:

  • How to remove the fractional part of a number.
  • How to find the remainder of a division.

The following sections explain these techniques, which may also give you some clues as to how to solve the problem.

Removing the fractional part of a number

In the number 3.1415927, the integer part of the number is 3 and the fractional part of the number is .1415927.

Javascript has a function that will remove the fractional part of a number called Math.floor. For example, after the following statements are executed, the variable numberWithoutFraction will have the value 3.

var numberWithFraction = 3.1415927;
var numberWithoutFraction = Math.floor(numberWithFraction);

As you might guess, there is also a "ceiling" function (Math.ceil) You can learn more about these functions here .

Find the remainder of a division

You may recall from math that if you divide one number by another number you may have a remainder (the part of the first number that could not be divided evenly by the second number).

In Javascript, we can divide two numbers using the symbol /. For example, after the following statements are executed, the variable numberWithFraction will have the value 2.5.

var firstNumber = 5;
var secondNumber = 2;
var numberWithFraction = firstNumber / secondNumber;

In a way similar to division, Javascript uses the symbol % for remainder. After the below statements are executed, the variable remainder will have the value 1.

var firstNumber = 5;
var secondNumber = 2;
var remainder = firstNumber % secondNumber;

The value 5 can be divided evenly by 2 twice, but 1 is left over (the remainder).
