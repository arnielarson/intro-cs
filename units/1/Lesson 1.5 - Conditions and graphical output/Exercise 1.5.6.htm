Blinking

When animating, we don't have to limit ourselves to changes in location. We can change anything we are drawing. For example, if we alternate between two colors for our circle, it will appear to blink.

We need a variable

Lets say we want to alternate between a light and a dark color. We can have a variable, called dark, which keeps track of whether we should draw a dark circle or a light one. The value of this variable will be true for one frame, false for the next, then true and so on.

Flipping the lights on and off

We will want to change the value of the dark variable each frame. In english, we want to do the following every frame:

If the value of the dark variable is true, then set it to false.
If the value of the dark variable is false, then set it to true.

But since the dark variable can only ever true or false, this is the same as:

If the value of the dark variable is true, then set it to false.
Otherwise, set it to true.

We can use an If Statement to write this down.

if (dark) {
    dark = false;
} else {
    dark = true;
}

That's the only way to do it... not

Javascript has an easier way to do this. We can write:

dark = !dark;

The ! means not, which gives true if false, and false if true.

You can think of the ! giving the opposite boolean value.

Drawing the two colors

We will also need to change the code that draws the circle to either use a light color or a dark color. We can just set the fillStyle before we draw the circle. For example:

if (dark) {
    context.fillStyle = "green";
} else {
    context.fillStyle = "white";
}

Time to blink

Starting with the files from the last exercise, create a folder for this exercise (Exercise 1.5.6). Make the changes highlighted below:

var dark = true;

function drawSomething(context) {
    ...
    Set fillStyle before drawing circle
    if (dark) {
        context.fillStyle = some dark color;
    } else {
        context.fillStyle = some light color;
    }
    ...
}

function changeSomething(canvas) {
    dark = !dark;
    ...
}

Try it out. The circle should blink as it moves.
