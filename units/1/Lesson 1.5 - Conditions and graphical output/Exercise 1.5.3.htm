Conditions

Jumping around

So far in this class, we have learned about three ways we can control how our functions work.

We can either write our statements one after another, and the computer will execute them in the order we write them:

Or we can call a function, which puts the current function "on hold", and starts another function.

Or we can return from a function, which stops the current function and gives an answer back to the function that called it.

But what if we sometimes want to do one thing, and sometimes do another thing? What if we want to make a decision?

In this exercise, we are going to learn about conditions, which will allow us to make decisions in our functions.

Am I in Seattle? Or Winthrop?

In order to correct this function, we need to use the correct tax rate for the city. For Winthrop, the rate is 0.078. So we can add that rate.

But that still isn't right. We want to pick which rate to use based on what city (Seattle or Winthrop).

The city parameter tells us what city we are in. In order to pick the right rate, we need to compare the value of city to see which city we are in. But how do we compare values?

Thinking "is a equal to 5?", and then writing a = 5 instead of a == 5 is a common programming mistake. When your program runs, it may change the value of a instead of comparing 5 with it!

Boolean values

As you might imagine, the result of a making a comparison in Javascript is a value which is either true or false. This is called a boolean value, and is written in Javascript as true and false.

The word boolean is a reference to George Boole, an English Logician.

The following are examples of comparing against different values of the variable a.

Comparison When a is 4 When a is 5 When a is 6
a < 5 true false false
a <= 5 true true false
a > 5 false false true
a >= 5 false true true
a == 5 false true false
a != 5 true false true

Which rate?

Now we know when we are in Seattle (inSeattle is true), and when we are in Winthrop (inWinthrop is true).

When we are in Seattle, we want the rate to be assigned to 0.095.

When we are in Winthrop, we want the rate to be assigned to 0.078.

So we really need a way to pick between these two rates based on the values of inSeattle and inWinthrop.

If statement

Javascript has an If statement that we can use for this situation.

Read about the If Statement here.

Written in english, here is what we want our if statement to do:

If we are in Seattle, use a rate of 0.095.
If we are in Winthrop, use a rate of 0.078.
Otherwise, we don't know what rate to use.
