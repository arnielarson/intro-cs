Exercise 1.5.1

Canvas

Do your pages have drawings?

So far our pages have have had text and images on them. In this lesson, we will be working with the HTML Canvas Element which allows us to "draw" whatever we want within an area of the page.

Read the Introduction in the Canvas Reference. Come back to this page when you get to the END OF INTRODUCTION.

Getting Started

Download this zip as a starting point for this Exercise (Exercise 1.5.1).

The folder includes an HTML page with a named canvas element, an empty CSS file, and the following Javascript. You can get started drawing by filling in the drawSomething function.

function drawSomething(context) {    
    make your changes here
}

function start() {
    var canvas = document.getElementById("mainCanvas");
    var context = canvas.getContext("2d");
    drawSomething(context);
}

Change the drawSomething function to draw a Rectangle (either filled, stroked, or both).

Try changing the Colors And Line Width of your rectangle.

The assignment

Your assignment is to come up with something you want to draw, and then use canvas to draw it. Use the Canvas Reference as your guide.

It might be helpful to sketch what you want using paper/ink first, or find a picture you are going to base your drawing on.

Your drawing must contain:

  • At least one Rectangle
  • At least one Arc or Circle
  • A least one Line, Polyline or Polygon
  • At least one Text
  • At least one Image

For example, you drawing might contain: one text, two images, one arc, a filled background (rectangle), and a three lines.
