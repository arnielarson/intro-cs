Another dimension

Up until this point, our circle has been moving from left to right (horizontally) and bouncing. Let's make it move vertically as well (and bounce).

In english, we want to do the following for every frame:

Add dx to the value of x.
If x has reached the right edge of the canvas then negate the value of dx.
If x has reached the left edge of the canvas then negate the value of dx.
What else? We need to something similar for the vertical direction. This is for you to figure out.

Starting with the files from the last exercise, create a folder for this exercise (Exercise 1.5.5). Make changes so that the circle moves vertically and horizontally, and bounces when it reaches the edge of the canvas. The top of the canvas is 0. The bottom of the canvas is canvas.height.

If your circle is taking a too-predicable path, use a vertical change of 3, and a horizontal change of 7.
