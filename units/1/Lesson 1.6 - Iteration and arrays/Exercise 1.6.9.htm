Exercise 1.6.9

If you finish

If you finish the other exercises, here are some ideas for additional exploration.

Growing bubbles

Make each bubble grow a little each frame, and if they reach a certain size, then pop them.

Increase the radius of each bubble each frame.

Each frame, check if the radius is too large, and if so, remove the bubble from the array.

Limited drift

Allow the bubbles to drift, but only for N frames (say 10).

Add a property to each bubble to keep track of how many frames a bubble has been around.

Add one to the property each frame.

Only change dx and dy if the bubble has been around for less than say, 10 frames.

Warp speed

Hit warp speed in a bubble factory.

Start each bubble at a random point in the canvas.

Each frame, calculate dx and dy so that they move the bubble closer to the center.

When the bubble reaches approximately the center, remove it.

Gravity

Add some gravity.

Start each bubble 1/4 of the way down the canvas.

Each frame, add 1 to dy, to simulate gravity.

Colors

Change the color based on how long the bubble has been around.

Add a property to each bubble that counts how many frames it has been around.

Add one to the property each frame.

Add an array of color names.

Compute the index of the color to use from the colors array based on the frame count for the bubble. The larger the frame count, the larger the index.

Done!
