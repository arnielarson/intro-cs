Exercise 1.6.4 -20 Questions

A Simple Web Game

I'm thinking of a number between 1 and 100.
If you make a guess I'll tell you if you're above or below my number.
How many guesses do you think it will take you to guess correctly?

Assignment

Make this game using HTML and Javascript!

  • Initialize the game with a random guess

  • Interact with a user to play the game

  • Each time the user enters a guess - compare the guess to the number and report this to the user.

  • When the user enters the number report back that guess is correct.

  • Optional ChallengeGive the user only 10 guesses, and if the user has not guessed correctly on the 10th try, they lose!
