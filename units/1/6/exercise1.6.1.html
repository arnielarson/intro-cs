Thinking "is a equal to 5?", and then writing a = 5 instead of a == 5 is a common programming mistake. When your program runs, it may change the value of a instead of comparing 5 with it!

Boolean values

As you might imagine, the result of a making a comparison in Javascript is a value which is either true or false. This is called a boolean value, and is written in Javascript as true and false.

The word boolean is a reference to George Boole, an English Logician.

The following are examples of comparing against different values of the variable a.

Comparison When a is 4 When a is 5 When a is 6
a < 5 true false false
a <= 5 true true false
a > 5 false false true
a >= 5 false true true
a == 5 false true false
a != 5 true false true

Notice that comparing strings is case sensitive, meaning that whether the letters are upper or lower case matters ("cat" is different than "Cat").

Which rate?

Now we know when we are in Seattle (inSeattle is true), and when we are in Winthrop (inWinthrop is true).

When we are in Seattle, we want the rate to be assigned to 0.095.

When we are in Winthrop, we want the rate to be assigned to 0.078.

So all we need is a way to pick between these two rates based on the values of inSeattle and inWinthrop.

If statement

In Javascript we can use the If statement for this situation.
