Conventions

To keep our work readable by others (including your teachers), we are going to adopt a set of Conventions when writing HTML. You may want to take a look at them now.

Many software developers have some set of conventions that they follow that are more restrictive than the grammar of the language. Consistency is important when working in teams.

After reading the conventions, you might notice that the template that Komodo Edit provided is not following our conventions (for example, the indenting is not right). Lets fix that now. How do I write a less than character?

Say we wanted to write a mathematical expression like the following in HTML. How would we do it?

0 < a < 10

HTML is already using the less than symbol (<) to indicate the start of a tag. So how do we say we want the < to just be treated as text instead of a tag?

It turns out that HTML has a way to do this. When we want to use a character that is already being used by the language, the language can give a different way of writing it.

Take a look at the Character References in our HTML Subset. Use the syntax provided there to display the mathematical expression above instead of Jello. Save and reload in Chrome to make sure it matches the mathematical expression above.

If you get stuck, move your mouse here to see the answer.

0 &lt; a &lt; 10

Command-click (Mac) or Right-click (Windows) on the smiley below to get the menu for the image. Choose "save image as" and save the image to your exercise folder as smiley.png.

Be careful using images that you found on the web, since their use may be restricted.

An Image Element (abbreviated img) will display an image on a page. But which image?

The Image Element has an Source Attribute which controls which image to display. In this case, we can just use the name of the image file (smiley.png) as the value for the attribute. Our image element is then:

<img src="smiley.png">

Notice the Image Element has start tag (<img>), but no end tag (</img>). This is because an Image Element is not a container for other elements or text.
