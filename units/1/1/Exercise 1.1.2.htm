Exercise 1.1.2

The following are additional ideas for exploration. You don't have to do them in order. They are all optional.

You probably want to create a new folder (Exercise 1.1.2).

Create your own web site

Take the concepts you learned from Exercise 1.1.1 and build a bigger web site. It does not need to be related to smileys or math in any way!

Try more HTML elements

Try creating a Anchor Element to create a link that is within a single page. You will need to make sure there are enough elements on the page if you want to see the page scroll when you click on a link.

Try creating a Header Element.

Try creating a Division Element.

Try creating a Span Element.

Use Developer Tools to inspect a web site

Point Chrome at http://www.universityprep.org/, and open Developer Tools.

Try to find the resources that the page uses.

Once Chrome has loaded the page, you can explore all the attributes for an element. For example, you can select the Image Element for Unversity Prep (top left of page), and inspect it's attribute values (look under Properties):

Find the URL of another image on the web. You can find the URL of an image on a page by choosing Command-click (Mac) or Right-click (Windows) on the image, and choose "Copy Image URL".

Change the value of the Source Attribute (abbreviated src) to the URL of the other image. You can double-click on the attribute to start editing, then paste the URL between the quotes. Press enter when you are done. What happened to the page?

Try changing other attribute values and see what happens to the page.
