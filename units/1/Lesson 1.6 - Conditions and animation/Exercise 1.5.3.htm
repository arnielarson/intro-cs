Exercise 1.5.3

Class size change

The UPrep staff has an additional question that they would like you to answer. They are interested in how much each class has changed in size from last year to this year.

For example, if the 11th grade had 80 students last year, and the 12th grade has 76 students this year, then the class decreased by 4 students (since 11th graders last year are 12th graders this year).

They would like you to calculate for 7th grade through 12th grade whether the class has increased or decreased by more than 4 students.

Calculating the change

We should be able to calculate the change for each grade by subtracting the value for this year from the corresponding value from last year.

That means an increase is a positive number, and a decrease is a negative number.

Checking the change

So if we know the change value, we can calculate whether the value has decreased or increased by more than 4 by writing:

var increasedByMoreThan4 = change > 4;
var decreasedByMoreThan4 = change < -4;

This will give us two values that are either true or false. We want a way to calculate whether either value is true.

Logical OR

Javascript has way of calculating whether either of two boolean values are true, called a logical OR expression. A logical OR expression is written:

firstExpression || secondExpression

The result of calculating a logical OR expression is true if either firstExpression or secondExpression are true, and false otherwise.

Just like a logical AND expression, a logical OR expression can be used more than once in an expression. For example, to see if three expressions are all true, we would write:

firstExpression || secondExpression || thirdExpression

Just like a logical AND expression, Javascript may not need to calculate all the expressions in a logical OR expression. As soon as one of the calculated expressions produces true, the overall expression is true, and Javascript can stop.

For example, if calculating firstExpression produces false, it will continue to secondExpression. If calculating secondExpression produces true, then it will stop (since there is no point in calculating thirdExpression).

Once again, you probably use this technique all the time without even thinking about it. Imagine you wanted to know if anyone in your class ate breakfast this morning. You could ask the first person, and they might say no. Then you could ask another person, and they might say yes. Would you continue asking more people? Probably not, since you know there is at least one person that ate breakfast (meaning someone ate breakfast). This is another example of Short Circuit Evaluation.

Using a logical OR expression

We can now use a logical OR expression to calculate whether a change is greater than 4.

var increasedByMoreThan4 = change > 4;
var decreasedByMoreThan4 = change < -4;
var changedByMoreThan4 = increasedByMoreThan4 || decreasedByMoreThan4;

Combining some of the expressions, we have:

var changedByMoreThan4 = change > 4 || change < -4;

The output

The output for this exercise should be an object value with a property for grade7 through grade12.

Each property should have a boolean value that is true if the change was greater than 4, and false otherwise.
