Exercise 1.3.2

Slope of a line using objects

To practice working with objects, lets update the program that calculated the slope of a line in the last lesson (Exercise 1.2.4) to use objects.

The assignment

  • You can start with what you did for Exercise 1.2.4.

  • Update your test function that calls the slope function:

    • Create object values for each of the points
    • Each object value will have two properties: x and y
    • Call your slope function with object values (instead of numbers)

  • Update your slope function that finds the slope of a line:

    • Update your function to have two parameters (one for each point)
    • Change the calculation to use the property values x and y of each point

  • Check the answers by running your program in Chrome.

    • Point (1, 2) to point (3, 4) has a slope of 1
    • Point (4, 3) to point (2, 1) has a slope of 1

  • Save your work, since you will need to submit it to schoology.

Done!
